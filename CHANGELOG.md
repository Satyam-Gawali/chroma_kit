## 1.0.0

* Initial release of ChromaKit.
* Added advanced color extensions: `withAlphaFactor`, `faint`, `contrastColor`, and `isDark`.
* Implemented safe hex parsing from strings with shorthand support (e.g., #FFF).
* Included shade manipulation utilities: `darken` and `lighten`.
* Built-in error handling for invalid hex inputs to prevent application crashes.
